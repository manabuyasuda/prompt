# prompt
Boilerplate of Pug, Sass(scss) and ES2016(Babel) by gulp.

## Requirement
- Node.js

## Get started
`npm install`

```
npm install
```

## Usage
Development task

```
gulp
```

or

```
npm run develop
```

build styleguide

```
gulp styleguide
```

or

```
npm run styleguide
```

publishing task

```
gulp htdocs
```

or

```
npm run htdocs
```
